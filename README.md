<h2>🛠️ Installation Steps:</h2>

<p>1. Clone Repo</p>

```
git clone https://www.github.com/adharvarun/Brainstorm.git
```

<p>2. Open Folder</p>

```
cd Brainstorm
```

<p>3. Install Dependencies</p>

```
pip install -r requirements.txt
```

<p>4. Run the App</p>

```
python manage.py runserver
```
